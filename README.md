# University-Landing-Page
Responsive landing page for PIEAS (Pakistan Institute of Engineering and Applied Sciences), showcasing programs, events, news, and institute highlights using HTML, Tailwind CSS, and JavaScript.


PIEAS Landing Page

This repository contains the landing page for PIEAS (Pakistan Institute of Engineering and Applied Sciences), Islamabad. The page is built using HTML, Tailwind CSS, and Font Awesome, featuring a responsive design, carousel, scroll animations, and sections for events, news, and quick links.

Preview

You can preview the landing page locally by opening index.html in your browser.

Features

Responsive Design: Works on desktops, tablets, and mobile devices.

Hero Carousel: Auto-play image slider showcasing PIEAS highlights.

Scroll Reveal Animations: Sections reveal smoothly as you scroll.

Events Section: Highlights latest events with images and descriptions.

News & Updates Sidebar: Shows latest news with links.

Quick Links & Contact Info: Accessible resources for students and visitors.

Footer with Social Links: Connects to PIEAS social media platforms.

Technologies Used

HTML5

Tailwind CSS (via CDN)

Font Awesome (via CDN)

Vanilla JavaScript for carousel and scroll animations
